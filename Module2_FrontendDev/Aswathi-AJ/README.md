# Digital Nurture 5.0 - Module 2: Frontend Development

## Student Portal Web Application

This folder contains 10 hands-on frontend exercises built around a student portal using HTML, CSS, JavaScript, React, Angular, Vue, and API integration.

### Topics Covered
- HTML5 and CSS3 fundamentals
- JavaScript ES6+ and DOM manipulation
- Fetch API, async/await, and Axios
- React components, hooks, routing, and Redux
- Angular services, routing, and reactive forms
- Vue.js Composition API, router, and Pinia
- Accessibility and cross-browser improvements
- Centralized API state management

### Folder Structure
- `handson_01` to `handson_04` — static and JavaScript-based portal exercises
- `handson_05` — React student portal
- `handson_06` — React Router + Redux version
- `handson_07` — Angular student portal
- `handson_08` — Vue.js student portal
- `handson_09` — accessibility-focused version
- `handson_10` — advanced API/state management version

### Learning Outcome
By the end of this module, the student portal is built progressively across multiple frontend frameworks and concepts, showing how the same application can be implemented using different technologies and patterns.

### Quick Summary of the Hands-On Work
- Handson 01: Basic website structure using HTML5 and CSS3
- Handson 02: Responsive layout design with Flexbox and Grid
- Handson 03: Dynamic page interactivity with JavaScript and DOM
- Handson 04: Async data fetching using Fetch API and Axios
- Handson 05: React app setup with components, props, and hooks
- Handson 06: React routing and global state with Redux
- Handson 07: Angular components, services, DI, and forms
- Handson 08: Vue.js app structure with Composition API and router
- Handson 09: Accessibility audit and keyboard-friendly UI updates
- Handson 10: Advanced API integration and centralized state handling


### Comparison 2: Frontend State Management Across Frameworks (Hands-On 06, 07, 08 & 10)

| Framework | State Library | State Pattern | Key Features |
| :--- | :--- | :--- | :--- |
| React | Redux Toolkit | Centralized Immutable Store | Immer immutability, Async Thunks for API calls, explicit dispatch / selector paradigm. |
| Angular | NgRx | Reactive Observables Stream | Pure reducer functions, RxJS streams, side-effects isolated in Effects. |
| Vue.js | Pinia | Composition API Setup Store | Native Vue 3 reactivity refs, computed getters, storeToRefs destructuring without losing reactivity. |
